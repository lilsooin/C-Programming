### multidimensional array
- Two-dimensional array or Three-dimensional array
- In C#, Two-dimensional array is **int[,]**, In C ***int[][]**
```c#
// C#
int[,] table = new int[2, 3];

for (int i = 0; i < 2 ++i)
{
    for (int j = 0; j < 3; ++j)
    {
        table[i, j] = (i + 1) * (j + 1);
    }
}
```

```c
/* C */
int table[2, 3];

for (int i = 0; i < 2 ++i)
{
    for (int j = 0; j < 3; ++j)
    {
        table[i][j] = (i + 1) * (j + 1);
    }
}
```

### Two-dimensional array and one-dimensional array
```c
/* Two-dimensional array */
int buffer[3][2];
int i;
int j;

for (i = 0; i < 3; ++i) {
    for (j = 0; j <2 ; ++j){
        buffer[i][j] = 0;
    }
}
```

```c
/* one-dimensional array */
int buffer[3 * 2];
int i;
int j;

for (i = 0; i < 3; ++i) {
    for (j = 0; j <2 ; ++j){
        buffer[i * 2 + j] = 0;
    }
}
```
